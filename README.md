# Mapping-Bachmann
Visualizing works by Ingeborg Bachmann in the German and Austrian National Libraries

#  Project Description

According to the German National Library's catalogue's data Ingeborg Bachmann is the eighth most translated women writer, with 108 translations from 14 original works (see https://www.tandfonline.com/doi/full/10.1080/29940443.2026.2674686#d1e347). This project is dedicated to mapping her works (originals and translations) utilizing data from the Austrian National Library (ONB) in order to highlight her importance as a representative writer of German literature.

# Data source and extraction

The dataset has been extracted from the online catalogue of the Austrian National Library (ONB) on July 2nd 2026 applying the following filters: Verfasserin=Bachmann, Ingeborg; Bücher.
